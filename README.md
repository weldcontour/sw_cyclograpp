# SW Cyclograpp

Приложение для построения циклограмм точечной сварки по параметрам сварочного контроллера Tecna TE550.

## Описание

SW Cyclograpp — это инструмент для инженеров-сварщиков и технологов, позволяющий визуализировать сварочный цикл точечной сварки. Приложение строит графики зависимости тока и усилия от времени на основе введённых параметров, полностью моделируя логику работы контроллера Tecna TE550.

## Основные возможности

- **Ввод 18 параметров сварочного цикла** с диапазонами, соответствующими Tecna TE550
- **Построение циклограммы** с двумя независимыми графиками:
  - Ток (синяя линия, %)
  - Усилие (красная линия, бар)
- **Две оси** для корректного отображения разных единиц измерения
- **Всплывающие подсказки** с точными значениями в точках графика
- **Мгновенное обновление** графика при изменении параметров
- **Две вкладки** для удобной навигации: "Параметры" и "Циклограмма"

## Параметры сварочного цикла

| Параметр | Описание | Диапазон |
|----------|----------|----------|
| SQUEEZE 1 | Время сжатия электродов | 0.5 – 99.5 циклов |
| PRESSURE | Давление сжатия | 0.5 – 10.0 бар |
| FORGE DELAY | Задержка проковки | 0 – 99 циклов |
| FORG.PRESS. | Давление проковки | 0 – 10.0 бар |
| PRE-WELD | Время предварительного подогрева | 0 – 99.5 циклов |
| PRE-POWER | Мощность предварительного подогрева | 5 – 99 % |
| COLD 1 | Интервал между подогревом и сваркой | 0 – 50 циклов |
| SLOPE UP | Время нарастания сварочного импульса | 0 – 25 циклов |
| WELD | Время сварки | 0.5 – 99.5 циклов |
| POWER | Мощность сварки | 5 – 99 % |
| IMPULSE N. | Количество импульсов | 1 – 9 |
| COLD 2 | Интервал между импульсами | 0 – 50 циклов |
| SLOPE DOWN | Время спада сварочного импульса | 0 – 25 циклов |
| COLD 3 | Интервал между сваркой и термообработкой | 0 – 50 циклов |
| POST-WELD. | Время последующей термообработки | 0 – 99.5 циклов |
| POST-POWER | Мощность последующей термообработки | 5 – 99 % |
| HOLD TIME | Время сжатия электродов после сварки | 0.5 – 99.5 циклов |
| OFF TIME | Пауза до следующего цикла | 0 – 99.5 циклов |

## Логика работы

Приложение моделирует реальный физический процесс:

1. **Холостой ход (0-10 циклов)** — давление отсутствует, электроды движутся к детали
2. **Рост давления** — со скоростью 1 бар за 1 цикл до достижения PRESSURE
3. **Достижение PRESSURE** — в момент SQUEEZE 1
4. **Сварка** — ток проходит через деталь
5. **Проковка** — если активна (FORGE DELAY > 0 и FORG.PRESS. > 0), давление повышается до FORG.PRESS.
6. **Спад давления** — после HOLD TIME до 0 со скоростью 1 бар за 1 цикл

## Технологии

- **Flutter** — кроссплатформенный фреймворк
- **fl_chart** — библиотека для построения графиков
- **Clean Architecture + MVVM** — архитектурный подход

## Установка и запуск

### Требования

- Flutter 3.10.6 или выше
- Dart 3.0.0 или выше

### Установка

```bash
# 1. Клонируйте репозиторий
git clone git@github.com:weldcontour/sw_cyclograpp.git

# 2. Перейдите в папку проекта
cd sw_cyclograpp

# 3. Установите зависимости
flutter pub get

# 4. Запустите приложение
flutter run -d chrome
# или
flutter run -d macos
```
